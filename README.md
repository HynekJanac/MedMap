## Medikmap.cz
Medikmap.cz má za cíl poskytnout začínajícím (i pokročilým) medikům, mapu vybraných kvalitních internetových zdrojů, nástrojů a studijních tipů. Svou cestu si, ale každý musí najít sám. 
## Zapojit se
